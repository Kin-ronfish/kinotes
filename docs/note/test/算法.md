# 位运算

- `&` 与运算符，全为真，才是真

```js
4&5 = 4 // 0100 & 0101 = 0100
```

- `|` 或运算符，全为假，才是假

```javascript
4|5 = 5 // 0100 | 0101 = 0101
```

- `^` 异或运算符，异为真，同为假

```javascript
4^5 = 1 // 0100 ^ 0101 = 0001
```

- `~` 非运算符，取反

```javascript
~4 = -5 // ~0100 = 1011
```

- `>>` 右移，原数值除以移位乘以2，左移同理

```
4>>1 = 2 // 0100 = 0010, num / (count x 2)
```
# 阶乘

```javascript
function n(num) {
  let res = 1;
  for(let i = i;i <= num; i++){
    res *= 1;
  }
  return res;
}
```
